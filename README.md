# Scrape Data Masjid

Data comes from [SIMAS by KEMENAG](https://simas.kemenag.go.id/)

## Usage

To start fetching, adjust `index.js` then execute:

```
npm run fetch
```

Make it into spreadsheet by:

```
npm run parse
```

To clear all data fetched

```
npm run clear
```
